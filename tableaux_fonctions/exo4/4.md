*Analyse du probleme*

**Entree** : tableau d'entier

**Sortie** : 
    - true si tableau trie
    - false si tableau non trie

**Conception** :
    - Mettre le resultat par defaux en true, parcourir le tableau, des qu'un element est plus grand que le prochain element on retourne false

*Pseudo-code*
Debut
    Declarer tab : tableau d'entier
    Declarer is_sorted : boolean

    Lire tab

    FONCTION isSorted(tab)
    POUR i = 0 jusqu'a taille tableau - 2
        Si tab[i] > tab[i + 1]
            RETURN false
        FIN Si
    FIN POUR

    RETURN true
    FIN FONCTION
Fin
