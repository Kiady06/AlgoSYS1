# Fonction SUM()

**Entree** : tab d'entier

**Sortie** : 0 ou somme des elements

**Conception** : On parcourt avec une boucle pour i = 0 jusqu'a taille tableau -1 et on fait : somme = somme + tab[i]

*Pseudo-code*
Debut
    Declarer tab : tableau d'entier;
    Declarer rslt : entier;

    FONCTION sum(tab)
    Lire tab
    rslt = 0
    POUR i = 0 jusqu'a taille tableau
        rslt += tab[i]
    FIN POUR

    RETURN rslt
    FIN Fontion
Fin3

# mean 

*Analyse du probleme*

**Entree** : tab entier non nul

**Sortie** : moyenne

**Conception** : 
    - j'utilise ma fonction sum et je divise le resultat de sum () par la taille du tableau

*Pseudo code*
Debut
    Declarer tab : tableau d'entier

    FONCTION mean(tab)
    Lire tab;

    RETURN (sum(tab)/(taille(tab)))
    FIN FONCTION
Fin

# variance

*Analyse du probleme*

**Entree** : tab d'entier

**Sortie** : variance

**Concepetion** : 
    * Utiliser square() sur le tab puis faire la moyenne du nouveau tableau obtenu avec mean()
    * utliser utiliser mean() sur le tableau puis mettre le resultat au carre
    * faire la difference

*Pseudo-code*
Debut
    Declarer tab : tableau d'entier

    FONCTION variance(tab)
    Lire tab

    return (mean(square(tab)) - mean(tab) * mean(tab))
    FIN FONCTION
Fin
