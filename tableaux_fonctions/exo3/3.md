# Fonction SUM()

**Entree** : tab d'entier

**Sortie** : 0 ou somme des elements

**Conception** : On parcourt la boucle de l'element 0 a la fin en sommant les nombres a chaque fois

*Pseudo-code*
Debut
    Declarer tab : tableau d'entier;
    Declarer rslt : entier;

    Lire tab
    rslt = 0
    POUR i = 0 jusqu'a taille tableau
        rslt += tab[i]
    FIN POUR

    RETURN rslt
Fin

# variance

*Analyse du probleme*

**Entree** : tab d'entier

**Sortie** : variance

**Concepetion** : 
    * Utiliser square() sur le tab puis faire la moyenne du nouveau tableau obtenu avec mean()
    * utliser utiliser mean() sur le tableau puis mettre le resultat au carre
    * faire la difference

*Pseudo-code*
Debut
    Declarer tab : tableau d'entier

    Lire tab

    return (mean(square(tab)) - mean(tab) * mean(tab))
Fin
