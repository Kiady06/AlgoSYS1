**Analyse du probleme**

*Entree* : tableau d'entier

*Sortie* : min et max du tableau

*Conception* :
    - parcourir le tableau du debut a la fin, mettre le premier nombre dans un tab et remplacer si le chiffre est inferieur ou superieur au nombre selon utilisation (tab[0] pour min et tab[1] pour max)

**Pseudo-code**
Debut
    Declarer tab : tableau
    Declarer result : tableau

    Fonction getMinMaxInteger( tab)
    Lire tab;
    result = [tab[0], tab[0]]

    POUR i = 1 jusqu'a taille tableau
        SI result[0] > tab[i]
            result[0] = tab[i]
        SINON SI result[1] < tab[i]
            result[1] = tab[i]
        FIN SI
    FIN POUR
    return result
FIN Fonction
Fin
